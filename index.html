Heading 1

16pixels is a set of CSS rules to ensure consistent 16 pixels based typography. Pixels are used for font sizes and line heights, as well as for paddings and margins. Can be used as typography normalizer. No dependencies. Wrap your HTML content with class .col and you are done. You will also like Typeboost.css
